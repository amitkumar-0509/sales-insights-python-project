# 📊 Sales Data Analysis Project (Python)

## 📌 Project Overview
This project focuses on analyzing sales data to extract meaningful business insights. Using Python and powerful data analysis libraries, I explored customer behavior, product performance, and revenue trends.

The goal of this project is to answer key business questions and support data-driven decision-making.

---

## 🧰 Tools & Technologies Used
- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Seaborn

---

## 📂 Dataset
The project uses the following datasets:
- Orders Data
- Customers Data
- Products Data

---

## 🎯 Business Questions Answered

1. 📅 Which months had the highest and lowest revenue?
2. 📦 Which product categories are most profitable?
3. 👥 Which customer segment spends the most?
4. 🏆 Who are the top 10 customers by lifetime value?
5. 🌍 Which city had the highest revenue?

---

## 🔍 Key Insights

- Identified peak and low-performing months based on revenue trends
- Discovered the most profitable product categories
- Found high-value customer segments contributing maximum revenue
- Ranked top 10 customers by lifetime value (LTV)
- Highlighted the city generating the highest sales

---

## 📊 Data Analysis Workflow

1. Data Cleaning
   - Handling missing values
   - Removing duplicates
   - Formatting data types

2. Exploratory Data Analysis (EDA)
   - Revenue trends over time
   - Category-wise performance
   - Customer segmentation analysis

3. Data Visualization
   - Line charts for trends
   - Bar charts for comparisons
   - Heatmaps for insights
